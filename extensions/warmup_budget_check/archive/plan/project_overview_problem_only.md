# 建筑室内空间连贯性研究：项目概览

## 研究问题

**核心问题**：哪些室内空间要素及其组合方式影响人们对空间连贯性的感知？

***

## 研究目标（双重）

### 目标1：统计推断（理论）

* 识别显著影响连贯性的主要因子（每一个自变量的主效应）

* 发现因子间的交互效应（主要是二阶、视情况可能发展三阶）

* 量化效应大小及置信区间

* 计划采用贝叶斯方法，按照比如说后验的稳定程度、宽度等，决定阶段或是否增加被试等

* **输出**：科学论文（效应量、显著性、p值）

### 目标2：预测建模（应用）

* 训练贝叶斯预测模型（不追求预测的绝对准确性与确定性，计划的预测形式是一个可能的分布，以建议形式给予。例如：在当前的情况下，有99%的可能连贯性不足，若将桌子减少3个，有95%的可能性增加连贯性）

* 对任意室内配置（但符合逻辑或正常的取值范围）预测连贯性评分（区间估计）

* **输出**：设计决策支持工具

**为什么需要两个目标？**

* 目标1回答"为什么"（因果机制）

* 目标2回答"是什么"（预测未知配置）

* 传统方法只能做好其中一个，我们两者兼顾

***

## 核心困境

### 困境1：设计空间巨大，预算有限

```
6个因子 → 1200种可能配置
实际预算 → 只能测试700-800次（60-67%）

问题：
- 无法全部测试（时间/成本）
- 必须智能采样（不能随机）
- 需要在有限样本下完成双重目标
```

### 困境2：不知道哪些交互重要

```
可能的二阶交互：C(6,2) = 15个
可能的三阶交互：C(6,3) = 20个

问题：
- 事先不知道哪些真实存在
- 如果全部测试，每个只有几次观测（不精确）
- 如果不测试，可能遗漏重要效应
```

### 困境3：被试资源的限制

```
每人只能测试25次（疲劳效应）
被试的总数有限
需要多个被试估计群体效应（ICC）

问题：
- 被试间差异大（测量噪声）
- 批次效应（不同时间段测试）
- 需要在个体和群体层面同时建模
```

### 困境4：统计推断 vs 预测建模的冲突

```
统计推断需要：
✓ 关键组合的充分重复（估计精度）
✓ 因子水平均衡覆盖（正交设计）

预测建模需要：
✓ 空间均匀分布（避免大片空白）
✓ 边界充分采样（外推能力）

→ 两者采样策略不同，难以同时优化
```
