**λ是什么？**

- λ控制EUR-ANOVA采样器**探索交互的热情程度**
- λ = 0.0 意味着完全忽略交互，只关注主效应
- λ = 1.0 意味着交互和主效应同等重要
- Phase 2采用**渐进策略**：从 0.100 增长到 0.240

**为什么λ要从低到高增长？（新策略）**

- Phase 2前期：λ值较低 → **稳固阶段**，先获得准确的主效应估计
- Phase 2后期：λ值增长 → **探索阶段**，在稳固基础上探索交互效应
- 这避免了过早探索交互导致主效应估计不准的问题
- 符合"先简单后复杂"的科学探索原则
